# Playwright Java Automation Framework - OrangeHRM

This is a Playwright Automation Framework developed using Java, Maven, and TestNG for automating the OrangeHRM demo application.

---

# 🚀 Technologies Used

- Java
- Playwright Java
- Maven
- TestNG
- Git & GitHub
- VS Code

---

# 🎭 Framework Components

## BaseTest
Handles:
- Playwright initialization
- Browser launch
- Page setup
- Test setup & teardown

## Page Object Model (POM)
Separate page classes created for:
- Login Page
- Dashboard Page
- Admin Page

## Config Reader
Used for:
- Reading application URL
- Managing reusable configuration data
